# MATBEA Enterprise — Developer Guide

This is a single-page React landing page for "MATBEA Enterprise," a mock crypto exchange targeting institutional clients. There is no backend, no real API calls, and no real trading — all data is hardcoded constants at the top of `App.tsx`. The tech stack is React 19 + TypeScript 6, built with Vite 8, and every UI component comes from MUI v9 (`@mui/material` and `@mui/icons-material`). Styling is done via MUI's `sx` prop, a dark theme configured in `theme.ts` with `createTheme`, and global CSS classes in `index.css` (animations, glassmorphism, dot-grid background, marquee scroll). The entire application lives in four files: `main.tsx` (entry point, wraps App in ThemeProvider), `theme.ts` (dark palette, typography, component overrides), `App.tsx` (all sections as inline functions — Navbar, TickerTape, HeroSection, RateCalculator, MarketSection, TradingPairsSection, FeaturesSection, StatsSection, CTASection, Footer), and `index.css` (global styles and keyframes). The dev server is already running at `http://localhost:5173/bug-free-waddle/` — never start it manually. Use browser MCP tools to take screenshots when making UI changes. Run `npm run lint` (oxlint, 103 rules) before committing. Responsive design uses MUI's `useMediaQuery` and Grid size props; the mobile drawer replaces desktop nav links. The RateCalculator uses `useState` + `useMemo` with a `ratePairs` object holding mock exchange rates for 13 currencies — to add a new one, just add its symbol and rate there. The white "M" logo is an inline SVG component called `MLogo` in `App.tsx`. All animations are pure CSS keyframes with no animation libraries.
